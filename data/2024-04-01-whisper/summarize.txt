En este video, exploramos cómo transformar audio a texto con alta precisión utilizando Whisper, un modelo de inteligencia artificial de OpenAI que es de código abierto. Para llevar a cabo este proceso, utilizamos Python y trabajamos dentro de un entorno creado con Conda, un gestor de ambientes recomendado para proyectos de inteligencia artificial. Después de configurar nuestro ambiente, instalamos herramientas esenciales como Poetry para la gestión de dependencias y FFmpeg, una librería necesaria para el manejo de audio que Whisper utiliza.

Procedimos a instalar la librería de OpenAI Whisper en Python y, a pesar de que el proceso puede variar en tiempo dependiendo de la conexión a internet y las especificaciones de la computadora, logramos configurar todo para empezar a escribir nuestro código. Creamos un archivo `main.py` donde importamos Whisper y preparamos nuestro ambiente de desarrollo en Visual Studio Code, asegurándonos de estar en el entorno correcto.

Después, definimos una función para obtener el transcript de un audio, seleccionando uno de los modelos de Whisper según la precisión deseada. Whisper ofrece varios modelos, desde Tiny hasta Large, cada uno con diferentes requerimientos de memoria y tiempo de procesamiento. Elegimos un modelo y, mediante nuestra función, procesamos un archivo de audio de ejemplo para obtener su transcripción.

Finalmente, exploramos cómo exportar estas transcripciones en formatos útiles como TSV, TXT y SRT, mostrando cómo estos formatos pueden ser utilizados para diferentes propósitos, como subtitulación o análisis de datos. También menciono que, aunque en este tutorial corremos los modelos localmente, estos pueden consumir bastante RAM y, normalmente, funcionan mejor con acceso a una GPU. Sin embargo, hay opciones para ejecutar estos modelos en la nube a través de proveedores como Azure, Cloudflare, o directamente con la API de OpenAI, lo que simplifica el proceso y elimina la necesidad de una configuración local compleja.
