Google Chrome está explorando una innovadora integración de inteligencia artificial directamente desde el navegador mediante una nueva API llamada Windows.ai. Esta API permitirá interactuar con un modelo de lenguaje grande (LLM) sin necesidad de servidores externos, todo integrado dentro del propio navegador. Este desarrollo aún no está disponible en los navegadores de forma general, pero se pueden realizar configuraciones en Google Chrome Dev para habilitar y probar esta funcionalidad.

Para habilitar Windows.ai, es necesario instalar Google Chrome Dev y seguir una serie de pasos que incluyen activar ciertas configuraciones y reiniciar el navegador varias veces. Una vez habilitada, esta API permite realizar tareas como escribir un poema a través de comandos simples en JavaScript directamente en la consola del navegador. Sin embargo, para que la API funcione completamente, también es necesario descargar e instalar el modelo LLM específico, en este caso, Gemini Nano.

La principal ventaja de esta integración es la capacidad de ejecutar modelos de AI directamente en el dispositivo (on device), lo que mejora la privacidad y reduce la latencia al no tener que comunicarse con servidores externos. Además, esta funcionalidad podría eventualmente extenderse a otros sistemas operativos como Android, permitiendo una integración más profunda de AI en aplicaciones móviles.

El enfoque directo en el navegador plantea preocupaciones sobre la interoperabilidad, ya que esta API, por ahora, solo funcionaría en Google Chrome, limitando su uso en otros navegadores como Firefox o Safari. A pesar de esto, la propuesta de Google marca un paso importante hacia la integración más amplia de AI en la web y abre nuevas posibilidades para el desarrollo de aplicaciones interactivas y responsivas directamente desde el navegador.
